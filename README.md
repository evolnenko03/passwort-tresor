# Password Generator MVP

Kurzbeschreibung:
- Laravel 12 + Blade + Tailwind CSS v4 + Alpine.js
- Clientseitige Passwort-Generierung mit crypto-sicheren Zufallszahlen
- Route: `/passwords`

Start:
1. `npm install`
2. `npm run dev`

## Code-Struktur
- Die UI ist in Blade-Komponenten aufgeteilt, damit die View übersichtlich bleibt.
- Die Alpine-Logik liegt in `resources/js/passwords/generator.js`, um Zustände und Regeln zentral zu pflegen.
- Alles bleibt clientseitig, damit der Generator ohne Backend-Logik funktioniert.
